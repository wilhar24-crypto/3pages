Web Development Unit

We will learn the basics of web pages. Start with HTML tags, then add simple styles, then use Bootstrap to help with layout.

  • Page structure: doctype, html, head, body
  • Content: headings, paragraphs, links, images, lists
  • Styling: inline style vs classes
  • Bootstrap: spacing and grid

Quick Tag Reference

Tag What it does
h1 Main heading
p Paragraph (a block of text)
a Link to another page
img Shows an image
ul and li Bullet list and items
div Box to group content
